Visualizes the number of posts per month in a column chart.

== Description ==

This plugin adds a widget to the dashboard, visualizing the number of posts of recent months in a column chart. The default number of month to be displayed is 15 - this can be changed in the configurations.
The column chart can maximal display until the month of your first post.

== Installation ==

1. Upload all files to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Enjoy the plugin on the Dashboard
